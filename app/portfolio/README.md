# Portfolio Pages

This directory contains the portfolio section of the application.

## Structure

```
portfolio/
├── page.tsx          # Portfolio listing page
└── [slug]/           # Dynamic portfolio item pages
    └── page.tsx      # Individual portfolio item page
```

## Purpose

The portfolio section is intended to showcase:

- Project listings with thumbnails
- Detailed project information
- Technologies used
- Live demos and source code links
- Project categories and filters
